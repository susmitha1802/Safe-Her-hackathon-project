# Safe-Her-hackathon-project
SafeHer is a women's safety web application that enables users to store trusted contacts and instantly send emergency alerts with location for faster help.
SafeHer - Women Safety Emergency Web App
->Users can save their basic details and trusted contact numbers.
->When and emergency happens,it sends an alert along with the user's current location to the saved contacts so they can help quickly.
->However,features like calling contacts,sending real alert messages, live safety path tracking, and real emergency communication are not fully implemented and are only     part of the demonstration.
  The main goal of this project is to show the user interface,workflow,and logic of a safety system.
  The AI Legal Assistant included in this is functional and works well.
